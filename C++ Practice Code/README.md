# DUKE ECE-551 C++ Practice Code

## 📚 项目简介

这是杜克大学ECE-551课程的C++学习实践代码库，包含了从C语言基础到高级C++编程的完整学习路径。项目按照学习进度分为多个阶段，每个阶段都包含相应的练习代码和实例。

本项目旨在通过系统性的学习路径，帮助学习者从零基础开始，逐步掌握C/C++编程的核心技能，最终具备独立开发高质量软件项目的能力。

#### 🔥 核心技能目标
- **编程基础扎实**：掌握C/C++语言的核心语法和编程思想
- **系统编程能力**：具备Linux环境下的系统级编程技能
- **面向对象精通**：深入理解OOP思想，熟练运用设计模式
- **高性能编程**：掌握内存管理、多线程、网络编程等高级技术
- **工程实践能力**：具备完整项目开发、测试、部署的工程化能力

#### 🚀 职业发展目标
- **技术深度**：成为C++领域的技术专家
- **解决问题**：具备分析和解决复杂技术问题的能力
- **团队协作**：掌握现代软件开发流程和团队协作技能
- **持续学习**：建立终身学习的技术成长体系

## 🎯 具体学习目标

### 阶段一：C语言基础 (1_C) - 17天系统学习

**核心目标：** 建立扎实的C语言编程基础，为后续C++学习打下坚实基础

#### 第1-2天：语言基础
- **宏和预处理器**：掌握#define、#include、条件编译等预处理指令
- **变量和数据类型**：理解基本数据类型、变量声明、作用域概念
- **输入输出**：熟练使用printf/scanf格式化输入输出
- **类型转换**：掌握隐式和显式类型转换规则

#### 第3-4天：运算符和控制结构
- **运算符系统**：算术、关系、逻辑、位运算符的使用
- **位运算精通**：掌握位操作技巧，解决经典面试题
- **控制流程**：if-else、switch-case、循环结构
- **数组操作**：一维、多维数组的声明、初始化和操作

#### 第5天：函数和递归
- **函数设计**：函数定义、声明、参数传递机制
- **变量作用域**：局部变量、全局变量、extern关键字
- **递归算法**：递归思想、经典递归问题（汉诺塔、八皇后）
- **项目实践**：掷骰子游戏、扑克牌发牌算法

#### 第6-8天：指针和字符串
- **指针核心概念**：指针的本质、指针运算、指针安全
- **指针高级应用**：指针与数组、函数指针、指针作为返回值
- **字符串处理**：字符串常量、字符串变量、string.h库函数
- **结构体编程**：struct定义、typedef、枚举类型
- **动态内存管理**：malloc/free的正确使用

#### 第9-11天：数据结构基础
- **动态数据结构**：动态数组、动态结构体
- **链表实现**：单向链表的完整实现（增删改查）
- **栈和队列**：基于数组和链表的栈、队列实现
- **二级指针**：指针的指针概念和应用

#### 第12-15天：高级数据结构和算法
- **哈希表**：哈希函数设计、冲突处理、性能分析
- **二叉搜索树**：BST的插入、删除、查找、遍历
- **排序算法**：冒泡、选择、插入、快速、归并排序的实现和分析
- **查找算法**：线性查找、二分查找及其变种
- **文件操作**：文本文件和二进制文件的读写

#### 第16-17天：综合应用
- **项目整合**：综合运用所学知识完成复杂项目
- **编译原理入门**：简单词法分析器的实现
- **代码优化**：性能分析和优化技巧

### 阶段二：Linux系统编程 (2_Linux) - 系统级编程能力

**核心目标：** 掌握Linux环境下的系统编程，理解操作系统原理

#### 库和构建系统 (Linux05)
- **静态库和动态库**：库的创建、链接和使用
- **Makefile编写**：自动化构建系统，依赖管理
- **编译链接过程**：深入理解编译、汇编、链接各阶段

#### 系统调用基础 (Linux06-08)
- **文件系统调用**：open、read、write、close等系统调用
- **文件描述符**：理解文件描述符概念，dup/dup2的使用
- **错误处理**：errno机制，系统调用错误处理

#### 进程编程 (Linux10-11)
- **进程基础**：进程创建（fork）、进程等待（wait）
- **进程间通信**：管道、命名管道、共享内存、消息队列
- **信号处理**：信号的发送、接收和处理

#### 多线程编程 (Linux12-13)
- **pthread线程库**：线程创建、同步、通信
- **线程同步**：互斥锁、条件变量、读写锁
- **生产者消费者模型**：经典并发编程模式

#### 网络编程 (Linux14-30)
- **Socket编程**：TCP/UDP套接字编程
- **网络协议**：深入理解TCP/IP协议栈
- **高并发服务器**：select、poll、epoll等I/O多路复用
- **进程池/线程池**：多种实现方案，性能对比分析

### 阶段三：C++基础 (3_CppBase) - 24天面向对象编程

**核心目标：** 从C语言过渡到C++，掌握面向对象编程思想和现代C++特性

#### C++入门 (Day01-03)
- **C到C++的转换**：命名空间、引用、函数重载
- **输入输出流**：iostream库的使用
- **内存管理**：new/delete操作符

#### 面向对象基础 (Day04-09)
- **类和对象**：类的定义、对象的创建和使用
- **构造函数和析构函数**：对象的生命周期管理
- **拷贝构造和赋值**：深拷贝vs浅拷贝，赋值运算符重载
- **运算符重载**：各种运算符的重载技巧
- **友元和静态成员**：友元函数、友元类、静态成员

#### 继承和多态 (Day10-13)
- **继承机制**：public、protected、private继承
- **虚函数和多态**：动态绑定、虚函数表
- **抽象类**：纯虚函数、接口设计
- **多重继承**：菱形继承问题、虚继承

#### 异常处理 (Day14)

- **异常安全**：RAII原则、异常安全保证

#### 模板编程 (Day15-17)
- **函数模板**：模板函数的定义和使用
- **类模板**：模板类的设计和实现
- **模板特化**：全特化和偏特化
- **模板元编程**：编译期计算

#### STL标准库 (Day18-20)
- **容器类**：vector、list、deque、set、map等容器
- **算法库**：sort、find、transform等算法
- **迭代器**：迭代器类型、迭代器适配器
